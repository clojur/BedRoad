使用"块"提高建模效率

工程应用

         许多工程的有限元模型，往往呈现出很高的重复性和对称性。例如：对称悬臂施工的变高梁、重复出现的横隔板、桥塔等。

         使用"块"不仅可以提高此类模型的输入效率，同时可以在修改模型时，通过"块"所具有的联动性，保证修改的效率和无误差输入。

         您可以把桥博中的块对象，就相像为AutoCAD中的块，但在桥博块不仅保存了单元的几何信息，还保存了节点、截面上的专业信息。

操作方式

         新建"段"对象 → 通过"段"创建"块"对象 → 插入"块"对象。

         ① 创建段

                   在工具条上，点击"建段"，并指定段的起始位置、终止位置：

        

         ② 创建块

                   在工具条上，点击"建块"，并选择需要纳入块中的段对象（选择表示段的标注）。确认后，为块起名。

        

         ③ 插入块

                   在工具条上，点击"插块"，指定插入点和插入方式(P代表平行插入、M表示镜像插入）
