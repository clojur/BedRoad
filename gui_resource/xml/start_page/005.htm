梁格自动划分

建模步骤：

CAD设定图层

         ① 在CAD中，将每个构件的轴线（一般为在平面上定义腹板的线）分别放在不同的图层中。

   

        

         ② 在桥博V4中，建立一个截面，并且为截面加上特征点信息。

         这些特征点可以将一个整体的箱梁截面划分为若干个子腹板截面，并且能够通过它来自动计算有效宽度。

         当梁格为变截面时，截面的本身的几何描述需要使用参数化的表达方式。
