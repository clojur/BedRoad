2、功能介绍

Ø  总体介绍：

n  《桥梁博士V4.0》具有强大的空间计算核心，面向桥梁全生命周期进行施工阶段分析、日常营运分析、撞击分析、地震分析、超载分析等。计算速度快、验算规范涵盖广，可以满足常规桥及斜拉桥、悬索桥等各种特殊桥梁的设计需求。

 

Ø  2.2 产品概念：

n  将力学分析功能与桥梁专业需求融洽结合，贴合设计规范，为桥梁设计人员提供系统、高效的分析设计工作平台。

 

Ø  单元类型：

n  空间变截面梁单元（考虑翘曲自由度）

n  杆单元

n  Ernst修正索单元：斜拉桥拉索

n  悬链线索单元：悬索桥主缆

n  线弹性连接单元：板式橡胶支座等

n  任意非线性弹性连接单元：单向支承

n  双线性塑性连接单元：减隔震支座，塑形铰

n  线性阻尼器单元：阻尼器

n  板壳和实体单元（V4.0仅试用）：箱梁精细分析

 

Ø  力学分析类型：

n  线性静力分析

n  影响线及影响面分析

n  屈曲分析：第一类稳定

n  模态分析：冲击系数、人行桥自振、地震等

n  反应谱分析：抗震

n  线性时程分析：抗震、人致震动

n  几何及材料非线性静力分析：抗震、大跨桥梁

n  几何及材料非线性时程分析：抗震、人致震动

 

Ø  辅助设计计算类型：

n  调束计算：快速钢束设计辅助工具

n  调索计算：快速索力设计辅助工具

n  主缆找形半解析计算：高效而稳定的主缆 

n  线型计算

n  索力自动优化计算：索力自动设计

 

Ø  约束及边界条件：                

n  单节点刚性约束（支持顶推法施工）

n  单节点空间耦合弹性约束

n  单节点双线性塑性约束：减隔震支座

n  主从约束

n  刚臂连接

Ø  面向桥梁的建模功能

在桥梁计算软件中首次引入了比较全面的构件的建模功能，建模及结构查看过程中可以不再使用单元、节点等有限元级别的概念

 

Ø  贴合设计需要的后处理功能

以设计人员比较习惯的方式展示计算结果，查看规范验算方便快捷

 

Ø  基础计算

n  是目前国内唯一能对结构基础进行全面验算的商业软件，能计算单桩承载力、桩身强度裂缝、承台强度、沉降等；

n  适用于各种桥梁基础，支持多种桩基础形式：基础形式：任意形状承台桩基础、扩大基础；

n  桩基类型：灌注桩、打入方桩、PHC管桩、钢管桩。

n  桩基形式：变截面桩、同一基础下长短桩
